jQuery Selectors Quiz

1. Which jQuery selector selects all <p> elements on the page?

a) $("p")
b) $("#p")
c) $(".p")
d) $("p#")

2. How do you select an element with the ID main-content?

a) $("main-content")

c) $(".main-content")
d) $("div.main-content")

3. Which selector selects all elements with the class highlight?

a) $(".highlight")
b) $("#highlight")
c) $("highlight")
d) $("div.highlight")

4. What does $("ul li") select?

d) $("checkbox:checked")

9. Which jQuery selector selects only <div> elements that have the class box?
